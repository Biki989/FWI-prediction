# 🔥 FWI Prediction

A Machine Learning web app that predicts the **Fire Weather Index (FWI)** using weather data.

🚀 **Live App:**
https://fwi-prediction-seven.vercel.app/

---

## ⚙️ Tech Stack

* Python
* Flask
* Scikit-learn (Ridge Regression)
* HTML/CSS
* Vercel

---

## 🧠 Model

* Algorithm: Ridge Regression
* Predicts FWI based on weather parameters

---

## 📂 Structure

```
FWI-prediction/
├── models/
├── notebook/
├── templates/
├── Application.py
├── requirements.txt
└── vercel.json
```

---

## 🚀 Run Locally

```bash
git clone https://github.com/Biki989/FWI-prediction.git
cd FWI-prediction
pip install -r requirements.txt
python Application.py
```
