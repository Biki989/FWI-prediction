A — About This System

This application is a Machine Learning web tool built with Flask that predicts the Forest Fire Weather Index (FWI) — a numeric rating of fire intensity used by fire management agencies across the world. The underlying model is a Ridge Regression algorithm trained on real atmospheric readings from two Algerian forest regions. Input values are normalized via a StandardScaler before inference, ensuring consistent and reliable predictions regardless of unit scale.
